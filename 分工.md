### 数据库配置说明

项目使用 Spring Data JPA，设置 `ddl-auto=update` 会自动建表，只需创建好数据库即可。

#### 方式一：MySQL 配置

1. 安装 MySQL 8.0+
2. 创建数据库：

```sql
CREATE DATABASE intelligent_helmet DEFAULT CHARACTER SET utf8mb4;
```

3. `application.properties` 配置：

```properties
# MySQL 数据库配置
spring.datasource.url=jdbc:mysql://localhost:3306/intelligent_helmet?useSSL=false&serverTimezone=Asia/Shanghai&characterEncoding=utf8&allowPublicKeyRetrieval=true
spring.datasource.username=root
spring.datasource.password=123456
spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver

# JPA 配置
spring.jpa.hibernate.ddl-auto=update
spring.jpa.show-sql=false
spring.jpa.properties.hibernate.dialect=org.hibernate.dialect.MySQL8Dialect
```

4. `pom.xml` 依赖（已包含）：

```xml
<dependency>
    <groupId>mysql</groupId>
    <artifactId>mysql-connector-java</artifactId>
    <version>8.0.33</version>
</dependency>
```

#### 方式二：PostgreSQL 配置

1. 安装 PostgreSQL 14+
2. 创建数据库：

```sql
CREATE DATABASE intelligent_helmet WITH ENCODING 'UTF8';
```

3. `application.properties` 修改为：

```properties
# PostgreSQL 数据库配置
spring.datasource.url=jdbc:postgresql://localhost:5432/intelligent_helmet
spring.datasource.username=postgres
spring.datasource.password=123456
spring.datasource.driver-class-name=org.postgresql.Driver

# JPA 配置
spring.jpa.hibernate.ddl-auto=update
spring.jpa.show-sql=false
spring.jpa.properties.hibernate.dialect=org.hibernate.dialect.PostgreSQLDialect
```

4. `pom.xml` 需要替换 MySQL 依赖为：

```xml
<dependency>
    <groupId>org.postgresql</groupId>
    <artifactId>postgresql</artifactId>
    <version>42.6.0</version>
</dependency>
```

#### 两种数据库对比

| 对比项 | MySQL | PostgreSQL |
|--------|-------|-----------|
| 驱动类 | `com.mysql.cj.jdbc.Driver` | `org.postgresql.Driver` |
| URL 格式 | `jdbc:mysql://localhost:3306/db_name` | `jdbc:postgresql://localhost:5432/db_name` |
| 方言 | `MySQL8Dialect` | `PostgreSQLDialect` |
| 默认端口 | 3306 | 5432 |
| 建库命令 | `CREATE DATABASE db_name DEFAULT CHARACTER SET utf8mb4;` | `CREATE DATABASE db_name WITH ENCODING 'UTF8';` |

> **注意**：由于使用了 JPA 的 `ddl-auto=update`，两种数据库都会自动根据 Model 实体类建表，不需要手动写建表 SQL。切换数据库只需改 properties 和 pom 依赖即可。
