# 答辩答案：接口与前后端代码对应关系

## 一、系统整体架构

```
浏览器 → http://localhost:5173/前端路由 → Vue 前端页面
                                         ↓ (axios 请求)
                                    http://localhost:8082/api/xxx → Spring Boot 后端 Controller
```

- **前端**：Vue 3 + Vite，开发端口 5173，通过 Vite 代理转发 `/api` 请求到后端 8082
- **后端**：Spring Boot，端口 8082，RESTful API
- **认证**：JWT Token，存储在 `sessionStorage`

---

## 二、HTTP 请求工具（前端公共模块）

**文件**：`intelligent-helmet-vue/src/utils/request.js`

```javascript
import axios from 'axios'

const request = axios.create({
  baseURL: '/api',    // 所有请求自动加 /api 前缀
  timeout: 10000
})

// 请求拦截器：自动带上 token
request.interceptors.request.use(config => {
  const token = sessionStorage.getItem('token')
  if (token) {
    config.headers.Authorization = `Bearer ${token}`
  }
  return config
})
```

**作用**：封装 axios，自动给每个请求附加 JWT Token，实现免手动传递认证信息。

### 接口 1：用户登录

| 项目 | 内容 |
|------|------|
| **浏览器地址** | `http://ip:8082/api/auth/login` |
| **请求方式** | POST |
| **前端页面路由** | `http://localhost:5173/auth` |

#### 前端代码

**文件**：`intelligent-helmet-vue/src/views/auth/Auth.vue`

```javascript
// 用户点击登录按钮后执行
const handleLogin = async () => {
  const res = await request.post('/auth/login', {
    username: loginForm.username,
    password: loginForm.password
  })
  if (res.data.success) {
    sessionStorage.setItem('token', res.data.token)  // 保存 JWT
    router.push('/app')  // 跳转到主页
  }
}
```

**作用**：收集用户输入的用户名密码，发送到后端验证，成功后保存 token 并跳转。

#### 后端代码

**文件**：`intelligent-helmet-terminal/src/main/java/com/demo/controller/AuthController.java`

```java
@PostMapping("/login")
public ResponseEntity<Map<String, Object>> login(@RequestBody Map<String, String> request) {
    String username = request.get("username");
    String password = request.get("password");

    Map<String, Object> response = new HashMap<>();
    if (userService.validateUser(username, password)) {
        User user = userService.findByUsername(username).get();
        String token = jwtUtil.generateToken(username, user.getRole());
        response.put("success", true);
        response.put("token", token);
        response.put("username", username);
        response.put("role", user.getRole());
    } else {
        response.put("success", false);
        response.put("message", "用户名或密码错误");
    }
    return ResponseEntity.ok(response);
}
```

**作用**：
1. 接收前端发来的用户名和密码
2. 调用 `userService.validateUser()` 验证密码是否正确
3. 验证通过后用 `jwtUtil.generateToken()` 生成 JWT Token 返回给前端
4. 前端拿到 token 后存入 sessionStorage，后续请求都会带上这个 token

  })

#### 后端代码

```java
@PutMapping("/profile")
public ResponseEntity<?> updateProfile(@RequestHeader("Authorization") String auth,
                                       @RequestBody Map<String, Object> body) {
    User user = resolveUser(auth);
    UserProfile profile = profileRepository.findByUserId(user.getId())
            .orElse(new UserProfile());
    profile.setUserId(user.getId());
    // 逐字段更新
    if (body.containsKey("nickname")) profile.setNickname((String) body.get("nickname"));
    if (body.containsKey("age")) profile.setAge((Integer) body.get("age"));
    // ...
    profileRepository.save(profile);
    return ResponseEntity.ok("OK");
}
```

**作用**：用户在个人资料页修改信息后，PUT 请求把新数据发给后端保存到数据库。

## 七、一个完整请求的生命周期（以登录为例）

```
1. 用户在浏览器访问 http://localhost:5173/auth
2. Vue Router 匹配到 /auth 路由，渲染 Auth.vue 组件
3. 用户填写用户名密码，点击登录
4. Auth.vue 调用 request.post('/auth/login', {username, password})
5. axios 拦截器拼接完整URL: http://localhost:5173/api/auth/login
6. Vite 开发服务器的 proxy 将 /api 请求转发到 http://localhost:8082
7. Spring Boot 的 AuthController 接收到 POST /api/auth/login
8. 验证密码 → 生成 JWT Token → 返回 { success: true, token: "xxx" }
9. 前端收到响应，sessionStorage.setItem('token', token)
10. router.push('/app') 跳转到主页面
11. 后续所有请求都自动带上 Authorization: Bearer <token> 头
```

## 八、答辩常见追问及回答

**Q：前后端是怎么通信的？**
A：前端用 axios 发送 HTTP 请求（GET/POST/PUT/DELETE）到后端的 RESTful API。开发环境通过 Vite proxy 代理解决跨域。实时数据通过 WebSocket 推送。

**Q：JWT 认证流程是什么？**
A：登录成功后端生成 JWT Token → 前端存入 sessionStorage → 后续请求在 Header 带 `Authorization: Bearer token` → 后端拦截器验证 token 有效性 → 解析出用户身份。

**Q：前端怎么知道调用哪个后端接口？**
A：前端通过 `request.js` 封装的 axios 实例发请求，baseURL 设为 `/api`，加上具体路径如 `/auth/login`，最终请求 `/api/auth/login`，对应后端 `@RequestMapping("/api/auth")` + `@PostMapping("/login")`。

**Q：数据库里存了哪些表？**
A：`user`（用户账号）、`user_profile`（个人资料）、`sensor_data`（传感器数据）、`emergency_contact`（紧急联系人）、`ride_session`（骑行记录）、`sport_checkin`（运动打卡）、`helmet_event`（告警事件）、`chat_message`（聊天记录）。
