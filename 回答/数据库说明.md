## 数据库中的表

| 表名 | 说明 |
|------|------|
| users | 用户账号 |
| user_profiles | 用户个人资料 |
| sensor_data | 传感器历史数据 |
| emergency_contacts | 紧急联系人 |
| helmet_events | 头盔事件记录（跌倒/慢速等） |

---

## 各表字段与查询命令

---

### 1. users — 用户账号

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| username | VARCHAR(50) | 用户名，唯一 |
| password | VARCHAR(255) | 加密密码 |
| role | VARCHAR(20) | 角色，默认 "user" |
| deviceId | VARCHAR(100) | 绑定设备ID |
| createdAt | DATETIME | 注册时间 |
| updatedAt | DATETIME | 最后更新时间 |

```sql
-- 查询所有用户
SELECT id, username, role, deviceId, createdAt FROM users;

-- 查询某个用户
SELECT * FROM users WHERE username = 'your_username';
```

---

### 2. user_profiles — 用户个人资料

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| user_id | BIGINT | 关联 users.id |
| nickname | VARCHAR(20) | 昵称 |
| age | INT | 年龄 |
| height | INT | 身高(cm) |
| weight | DOUBLE | 体重 |
| gender | VARCHAR(10) | 性别 |
| avatarData | TEXT | 头像（Base64） |
| bio | VARCHAR(200) | 个人简介 |
| weightUnit | VARCHAR(5) | 体重单位 |
| bloodType | VARCHAR(10) | 血型 |
| allergies | VARCHAR(200) | 过敏源 |

```sql
-- 查询所有用户资料（关联用户名）
SELECT u.username, p.nickname, p.age, p.height, p.weight, p.gender, p.bloodType, p.allergies
FROM user_profiles p
JOIN users u ON p.user_id = u.id;

-- 查询某用户资料
SELECT * FROM user_profiles WHERE user_id = 1;
```

---

### 3. sensor_data — 传感器历史数据

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| deviceId | VARCHAR(100) | 设备ID（有索引） |
| appId | VARCHAR(100) | 应用ID |
| temperature | DOUBLE | 温度(°C) |
| humidity | DOUBLE | 湿度(%) |
| longitude | DOUBLE | 经度（保留4位小数） |
| latitude | DOUBLE | 纬度（保留4位小数） |
| roll | DOUBLE | 横滚角 |
| pitch | DOUBLE | 俯仰角 |
| avm | DOUBLE | 加速度均值 |
| gvm | DOUBLE | 陀螺仪均值 |
| fallFlag | BOOLEAN | 跌倒标志 |
| slowFlag | BOOLEAN | 慢速标志 |
| receiveTime | DATETIME | 接收时间（有索引） |

```sql
-- 查询最新10条数据
SELECT * FROM sensor_data ORDER BY receiveTime DESC LIMIT 10;

-- 查询某设备最新数据
SELECT * FROM sensor_data WHERE deviceId = 'your_device_id' ORDER BY receiveTime DESC LIMIT 10;

-- 查询今天的数据条数
SELECT COUNT(*) FROM sensor_data WHERE DATE(receiveTime) = CURDATE();

-- 查询有跌倒事件的记录
SELECT * FROM sensor_data WHERE fallFlag = 1 ORDER BY receiveTime DESC;

-- 查询温湿度均值（按设备）
SELECT deviceId, AVG(temperature) AS avg_temp, AVG(humidity) AS avg_hum, COUNT(*) AS total
FROM sensor_data
GROUP BY deviceId;

-- 查询所有设备ID
SELECT DISTINCT deviceId FROM sensor_data;
```

---

### 4. emergency_contacts — 紧急联系人

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| user_id | BIGINT | 关联 users.id |
| name | VARCHAR(50) | 联系人姓名 |
| phone | VARCHAR(20) | 电话 |
| relation | VARCHAR(30) | 关系（如：父亲、朋友） |
| notes | VARCHAR(500) | 备注 |

```sql
-- 查询所有紧急联系人（关联用户名）
SELECT u.username, c.name, c.phone, c.relation
FROM emergency_contacts c
JOIN users u ON c.user_id = u.id;

-- 查询某用户的联系人
SELECT * FROM emergency_contacts WHERE user_id = 1;
```

---

### 5. helmet_events — 头盔事件记录

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| user_id | BIGINT | 关联 users.id |
| eventType | VARCHAR(30) | 事件类型（fall / slow） |
| eventName | VARCHAR(50) | 事件名称（如：摔倒事件） |
| deviceId | VARCHAR(100) | 设备ID |
| eventTime | DATETIME | 事件发生时间 |
| longitude | DOUBLE | 经度 |
| latitude | DOUBLE | 纬度 |

```sql
-- 查询最新20条事件
SELECT * FROM helmet_events ORDER BY eventTime DESC LIMIT 20;

-- 查询某用户的所有事件
SELECT * FROM helmet_events WHERE user_id = 1 ORDER BY eventTime DESC;

-- 按事件类型统计
SELECT eventType, COUNT(*) AS count FROM helmet_events GROUP BY eventType;

-- 查询跌倒事件
SELECT u.username, e.eventTime, e.longitude, e.latitude
FROM helmet_events e
JOIN users u ON e.user_id = u.id
WHERE e.eventType = 'fall'
ORDER BY e.eventTime DESC;
```

## 快速总览命令（进入 MySQL 后执行）

```sql
-- 查看所有表
SHOW TABLES;

-- 查看某张表的结构
DESCRIBE sensor_data;
DESCRIBE users;
DESCRIBE user_profiles;
DESCRIBE emergency_contacts;
DESCRIBE helmet_events;

-- 各表数据量
SELECT 'users' AS tbl, COUNT(*) AS cnt FROM users
UNION ALL SELECT 'user_profiles', COUNT(*) FROM user_profiles
UNION ALL SELECT 'sensor_data', COUNT(*) FROM sensor_data
UNION ALL SELECT 'emergency_contacts', COUNT(*) FROM emergency_contacts
UNION ALL SELECT 'helmet_events', COUNT(*) FROM helmet_events;
```
